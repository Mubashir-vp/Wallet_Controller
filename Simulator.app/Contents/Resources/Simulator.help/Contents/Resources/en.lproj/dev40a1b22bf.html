Simulate an external display

You can simulate an external display connected to an iOS device by choosing a display from the Hardware > External Displays menu.

The external display options are:

  • Disabled: Disconnects the external display.

  • [Screen resolution]: Opens a simulator of the external display with the specified resolution, or changes the external display to the resolution.

  • CarPlay: Opens a CarPlay simulator if a compatible iOS device is selected from the Hardware > Device menu.
