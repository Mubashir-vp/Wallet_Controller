Redirect keyboard shortcuts to a simulated device

You can send keyboard shortcuts that use the Shift, Command, Control, and Option modifier keys to a simulated device. Each simulated device can have a different setting for redirecting keyboard shortcuts and the setting is saved when the device is closed.

A device can detect if the user is pressing the left or right version of a modifier key while shortcuts are redirected.

Note: System shortcuts such as Command-Option-D are not redirected.

Redirect keyboard shortcuts to a device: Choose Hardware > Keyboard > Send Menu Keyboard Shortcuts to Device.

Stop redirecting shortcuts: Choose the menu item again or press both the left and right Command keys.

While keyboard shortcuts are being redirected the Send Menu Keyboard Shortcuts to Device menu item is checked and shortcuts are not displayed in Simulator menus.
