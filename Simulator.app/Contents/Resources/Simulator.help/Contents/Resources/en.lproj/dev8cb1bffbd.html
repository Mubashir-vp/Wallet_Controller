Remove simulator runtimes

Delete a simulator runtime for a version of an OS for a platform that you no longer need by opening /Library/Developer/CoreSimulator/Profiles/Runtimes/ and deleting the corresponding file. For example, to delete the simulator runtime for all iOS devices running version 8.3 of the operating system, delete the file iOS 8.3.simruntime.

WARNING: Quit Simulator, Xcode, and all other Xcode tools such as Instruments before removing simulator runtimes.
