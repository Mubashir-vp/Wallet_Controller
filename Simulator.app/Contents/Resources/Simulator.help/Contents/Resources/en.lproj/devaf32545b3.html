Test iCloud

Use Simulator to test if data is synchronized between iCloud and the simulated device. Use physical devices for performance testing.

  1. Sign in to iCloud on the simulated device.

    It is strongly encouraged that you create and use a separate Apple ID specifically for testing iCloud in Simulator.

  2. Choose Debug > Trigger iCloud Sync to trigger the system to synchronize updates with the server.

    Note: Synchronization must be triggered manually as Simulator does not support Push notifications.
