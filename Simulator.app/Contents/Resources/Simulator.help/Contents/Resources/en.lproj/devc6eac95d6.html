Set a location or route

Use Simulator to set a static location or route. You can also send locations and routes from your Mac to a simulator.

Choose a preset location

  • Choose one of the preset locations from the Location menu item in the Debug menu.

    Preset

    Behavior

    Apple

    Set the location to the Apple Infinite Loop campus in Cupertino.

    City Bicycle Ride

    Simulate the user taking a bicycle ride in Cupertino. The route repeats until the location is changed.

    City Run

    Simulate the user taking a run in Cupertino. The route repeats until the location is changed.

    Freeway Drive

    Simulate the user driving from Cupertino to San Francisco. The route repeats until the location is changed.

Choose no location

  • Choose Debug > Location > None to set the device to have no location.

Set a custom location

  1. Choose Debug > Location > Custom Location.

  2. In the dialog that appears, enter a latitude and longitude.

  3. Click the OK button to set the device location.
